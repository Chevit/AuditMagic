# Design: AuditMagic How-To Guide

**Date:** 2026-03-01
**Status:** Approved

## Summary

A task-based user guide written in Ukrainian for non-technical end users of AuditMagic. Delivered as `docs/HOW-TO.md` (Markdown), versioned in the repository.

## Format & Language

- Format: Markdown (`.md`)
- Language: Ukrainian
- Audience: Non-technical end users
- Location: `docs/HOW-TO.md`

## Structure

Task-based organization — each section answers "як зробити X?":

1. Що таке AuditMagic
2. Перший запуск
3. Як додати локацію
4. Як додати товар
5. Як знайти товар
6. Як змінити кількість
7. Як додати / видалити серійний номер
8. Як перемістити товар між локаціями
9. Як переглянути історію транзакцій
10. Як відредагувати або видалити товар
11. Як експортувати в Excel
12. Оновлення програми
13. Зміна теми

## Content Conventions

- Short numbered steps per task
- Bold for button/field names (e.g., **Додати**, **Зберегти**)
- Notes/warnings in blockquotes (`> ⚠️ ...`)
- No technical jargon (no "ORM", "SQLite", "FK", etc.)
